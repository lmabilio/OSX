UFS - Unix File System

Esse sistema de arquivos é hierárquico, distribuindo arquivos em uma árvore de diretórios. A base desta árvore é um diretório chamado diretório raizs. O UFS considera todos os dispositivos, discos, arquivos e diretórios como um todo alocando-os na mesma árvore de sistemas de arquivos. Assim, o usuário não precisa especificar o disco ou dispositivo onde estão os arquivos, somente o diretório armazenado. O próprio sistema operacional se encarrega de mapear os acessos ao disco ou dispositivo correspondente.
