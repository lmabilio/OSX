Baseado no Kernel Unix, OpenDarwin. o OS X (OS Ten) é um sistema operacional proprietário destinado exclusivamente aos computadores Mac e que combina a mesma GUI (Graphical User Interface ou Interface gráfica do usuário que permite a interação com elementos digitais através de elementos como ícones, etc.) desenvolvida para as versões anteriores do Mac OS com o núcleo derivado do micronúcleo Mach, um núcleo mais estável (Godoy e Pereira, 2008). Assim, o OS X, lançado inicialmente pela Apple Computer em 2001, é uma combinação do núcleo Darwin com uma renovada GUI chamada Aqua.
